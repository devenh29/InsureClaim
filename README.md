# 🚗 InsureClaim - AI Insurance Claim Processing System

An AI-powered system that automates insurance claim verification using OCR, document analysis, and fraud detection.

---

## 📌 Overview

InsureClaim allows users to submit insurance claims digitally by uploading required documents. The system processes and extracts data automatically and provides admins with a structured claim report.

---

## 🚀 Features

* 📄 OCR-based document extraction (DL, RC, Insurance, FIR)
* 🧠 AI-powered claim validation
* 🚨 Fraud detection system
* 📊 Admin dashboard with claim insights
* 🖼️ Damage image upload & preview
* 🔐 Role-based authentication (User/Admin)
* ⚡ Real-time backend (Firebase / Supabase)

---

## 🏗️ Workflow

User Upload → AI Processing → Data Extraction → Fraud Analysis → Admin Review → Decision

---

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS
**Backend:** Firebase / Supabase
**AI:** Python, OCR (Tesseract / Vision API)
**Database:** PostgreSQL
